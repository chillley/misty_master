### 项目结构


```
├── components                 # 通用组件目录
├── constants                  # constants 
├── fijkplayer_skin            # fijkplayer播放器皮肤（来自于abcd498936590）
├── model                      # model
├── pages                      # 源代码
│   ├── page_binding           # getx binding
│   ├── page_controller        # getx controller
│   ├── page                   # page
│   └── page_state             # state
├── routes                     # 路由文件
├── service                    # 接口文件配置
├── utils                      # 工具类
└── main.dart                  # main
```
